# Relatório Síntese - Atividade de Extensão UNIP
## DistriManager v1.0

**Projeto:** Sistema de gerenciamento de projetos desenvolvido em 85 horas por MazzarinDev. Combina aplicação desktop (Python/PyQt5) com plataforma web (React/Vite) para otimizar fluxo de trabalho de equipes.

**Funcionalidades:** Interface Kanban com datas fixas, persistência em JSON, tema Dark/Light, landing page profissional, dashboard com autenticação, upload JSON e gráficos interativos.

**Tecnologias:** Python 3.11, PyQt5, React 19, Vite, TypeScript, Tailwind CSS, Recharts.

**Objetivos:** 100% alcançados - Interface intuitiva, persistência de dados, página web moderna, autenticação, dashboard com métricas visuais, temas visuais e documentação técnica.

**Repositório:** https://github.com/MazzarinDev/DistriManager

**Status:** Concluído e pronto para produção.
